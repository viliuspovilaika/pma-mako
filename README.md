A stupid fast PhpMyAdmin brute-force tool

## Install

```sh
cargo install pma-mako
```

## Usage

```sh
./pma-mako <url (https://example.com/phpmyadmin/index.php)> <user_file> <pass_file> <thread_count> <server (usually 1)>
```
